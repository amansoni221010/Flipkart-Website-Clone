# Flipkart-Website-Clone

**Flipkart e-commerce website clone using react in which user can do the following:**<br>
Login/Signup<br>
Add item in the cart and place order<br>
See banner adds<br>
See product detail<br>

**Tools and Technnology used**:<br>
ReactJS<br>
NodeJS<br>
MongoDB<br>
